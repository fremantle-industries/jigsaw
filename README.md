# Jigsaw

A GUI frontend to manage the data processing engine from [`jig`](https://github.com/fremantle-industries/jig)

## Installation

Add `jigsaw` to your list of dependencies in `mix.exs`

```elixir
def deps do
  [
    {:jigsaw, "~> 0.0.1"}
  ]
end
```
